# Shift Control – Cold Storage Logistics

Website for **Shift Control**, a transportation-focused cold storage logistics company built with Vite and React.

## Pages
- Home – overview of our cold chain capabilities
- About – background on Shift Control
- Services – details of our logistics offerings
- Contact – get in touch with our team

## Quick start
```bash
npm install
npm run dev
# open http://localhost:5173
```

## Build
```bash
npm run build
# static files in ./dist
```

## Project
- Designed for modern deployment pipelines with GitHub and Hostinger.
